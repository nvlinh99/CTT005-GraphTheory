Quy định nộp bài: MSSV.py
---------------------------------------------------------------------------
Cài đật thuật toán graph coloring cho bài toán Sudoku
Input: Nhập vào N từ bàn phím.
Output: In ra một cách sắp xếp các chữ số thỏa mãn game Sudoku
---------------------------------------------------------------------------
Ví dụ

Input: N = 3
Output:
1 2 3
3 1 2
2 3 1
